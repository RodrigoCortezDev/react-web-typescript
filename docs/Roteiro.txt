- Criando projeto com:
    -> create-react-app NOME --template=typescript

- Instalando ESLINT para tratamentos de codigo:
    -> yarn add eslint -D
    -> yarn eslint --init   (quando perguntar se quer instalar com NPM colocar não e copiar as dependencias e mandar instalar com YARN)
    * criar o arquivo .eslint.ignore e alterar .eslintrc.json
    -> yarn add eslint-import-resolver-typescript -D  (Faz com que o Eslint entenda o typescript)

- Instalando o Pretier (formatador de codigo)
    -> yarn add prettier eslint-config-prettier eslint-plugin-prettier -D

- Instalando manipulador de rotas
    -> yarn add react-router-dom
    -> yarn add @types/react-router-dom

- Estilizaçao CSS com StyledComponents
    -> yarn add styled-components
    -> yarn add @types/styled-components -D

- Colocando a font ROBOTO
    *  Acessar o Google fonts, escolher os tipos de fontes e pegar o LINK e jogar abaixo do Title no Index.html dentro da pasta "public"
